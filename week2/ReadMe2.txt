# GitHub Python File Extractor

This script fetches a Python file from a GitHub page, extracts its code, saves it locally, and runs it.

## How to Use

1. Edit the `url` variable in the script to point to a raw GitHub file page.
2. Run the script:
   python extract_and_run.py

The script will:
- Extract Python code from the GitHub page
- Save it to `output.py`
- Execute the code

## Output

Creates `output.py` and runs it.
